# Inventivity Theme Child (v1.0.0) - MANIFEST

## Purpose
- A child theme for **Inventivity Theme** (Template: `inventivity-theme`).
- Includes your provided screenshot as `screenshot.png`.

## Files
- `style.css`  
  Child theme header (Template points to `inventivity-theme`) + place for custom CSS.
- `functions.php`  
  Enqueues parent stylesheet and child stylesheet.
- `screenshot.png`  
  Theme screenshot (your provided image).

## Install
1. Install and activate **Inventivity Theme** (parent).
2. Upload and activate **Inventivity Theme Child**.
